# SOC Packet Analyzer

## Overview
This project captures and analyzes network traffic using Python and Scapy.

## Features
- Packet sniffing
- IP extraction
- Real-time monitoring

## Tools
- Python
- Scapy
- Kali Linux

## How to Run
python3 -m venv venv
source venv/bin/activate
pip install scapy
sudo venv/bin/python analyzer.py
